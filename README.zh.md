App i18n
===

**App i18n** 是一个轻量级命令行工具，用于统一管理和优化多个应用的国际化流程。

它可以将 `.strings` 文件转换为结构化的 `.lproj` 目录，方便使用 AI 进行批量翻译；同时也支持将 `.lproj` 再转换回 `.strings` 文件，便于重新导入到 Xcode 项目中。

appi18n 专为需要规模化管理国际化资源的开发者设计，让翻译、整理和同步流程更加清晰高效。

### 为什么选择 appi18n？

* 🔁 支持 `.strings` ⇄ `.lproj` 双向转换
* 🤖 结构更适合 AI 自动翻译
* 📦 统一管理多个 App 的国际化文件
* ⚡ CLI 优先，便于脚本化集成
* 🛠 简洁、高效、可自动化

appi18n 让国际化流程变成一个清晰、可重复的自动化管道。
